## Result

Three benefits of using Git:

1. **Version history and rollback** - Git tracks every change to your codebase, allowing you to view the complete history of modifications and revert to any previous state if something goes wrong.

2. **Branching and parallel development** - You can create isolated branches to work on features or experiments without affecting the main codebase, then merge changes when ready. This enables multiple developers to work simultaneously without conflicts.

3. **Distributed collaboration** - Every developer has a complete copy of the repository, enabling offline work and providing redundancy. Changes can be shared and synchronized across team members through push/pull operations.
