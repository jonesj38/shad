## Result

Three benefits of using git:

1. **Version History and Rollback** - Git tracks every change made to your codebase, allowing you to view the complete history of modifications and revert to any previous state if something goes wrong.

2. **Branching and Parallel Development** - Git's lightweight branching model lets developers work on features, bug fixes, or experiments in isolation without affecting the main codebase. Branches can be merged back when ready.

3. **Distributed Collaboration** - Every developer has a full copy of the repository, enabling offline work and providing redundancy. Teams can collaborate through remotes (like GitHub/GitLab), with pull requests and code review workflows.
